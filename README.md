Pricing Page (Bootstrap Project)
This is a simple Pricing Page UI built using HTML and CSS and BOOTSTRAP.
It displays different pricing plans in a clean and responsive card layout.
Features:
->Responsive Navbar with toggle menu
->Pricing section with 3 plans:
->Free
->Pro
->Enterprise
->Bootstrap cards for structured layout
->Mobile-friendly design using Bootstrap grid
-> Clean and minimal UI

Technologies Used:
->HTML
->CSS3
->BOOTSTRAP 5
